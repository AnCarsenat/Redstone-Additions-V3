# Wiki Images

This folder contains all images used in the [GitHub Wiki](https://github.com/AnCarsenat/Redstone-Additions-V3/wiki).

## How to Add Screenshots

1. Take a screenshot in Minecraft (F2, or use a tool like ShareX)
2. Crop/resize to the recommended dimensions below
3. Save as `.png` in the appropriate subfolder
4. Push to the `main` branch — the wiki references these via raw GitHub URLs
